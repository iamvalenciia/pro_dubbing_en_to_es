### 3. Verify Optimization Applied
Look for these log lines:
```
[DIAR] ... batching=yes               ✅ Optimization enabled
[DIAR-BATCH] Batch optimization applied (batch_size=16)
```

```
set PYVIDEOTRANS_DIARIZATION_DEBUG=1
python run_ui_with_py310_engine.bat
```

## Configuration

### Environment Variables
```bash
# Override batch size (default: 16)
set PYVIDEOTRANS_DIARIZATION_BATCH_SIZE=16

# Enable debug logging
set PYVIDEOTRANS_DIARIZATION_DEBUG=1
```

### Recommended Batch Sizes
- **<24GB GPU**: 8-12
- **24-40GB GPU**: 16-20  
- **40-80GB GPU**: 24-32
- **>80GB GPU**: 32-64

✅ **Solution Implemented**: Batching wrapper processes 16 segments per GPU call
